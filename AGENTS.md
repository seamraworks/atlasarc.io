# AtlasArc Website Repo Notes

This repository contains the public AtlasArc.io website source.

- When this checkout sits inside a larger Seamra Works workspace, review the
  workspace-level notes before editing.
- In a standalone clone, this file and `README.md` are enough.
- For copy changes in a larger workspace, run the workspace-level publication
  checks before committing.
- Keep this repo focused on publishable website source and build assets. Leave
  drafts, planning notes, and experimental material outside this repo unless an
  explicit request says otherwise.

Local entry points:

- `README.md` for build commands and site architecture.
- `manifest/` for navigation, routes, and metadata.
- `src/` for page, content, style, and script source.
